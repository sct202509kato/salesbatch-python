# 売上集計ツール（Python）

## 概要
Excel / CSV ファイルを指定フォルダに配置し、  
ボタン操作（またはバッチ実行）で売上集計を自動生成するツールです。

日々の売上データ集計や月次レポート作成において、  
手作業による転記・集計ミスを減らすことを目的として作成しました。

## 主な機能
- Excel / CSV ファイルの一括読み込み
- 売上データの統合（複数ファイル対応）
- 商品別・月別の売上サマリー生成
- 集計結果を Excel（.xlsx）で自動出力
- 元ファイル名を保持し、データ追跡が可能

## 使い方（担当者向け）
1. `input` フォルダに Excel / CSV ファイルを入れる  
2. `run.bat` をダブルクリック（または UI から実行）  
3. `output/sales_summary.xlsx` が生成される  

※ Python の知識がなくても使用できることを想定しています。

## フォルダ構成
```
sales_batch/
├─ run.py # 売上集計ロジック
├─ run.bat # 実行用バッチ（1クリック実行）
├─ ui_streamlit.py # ローカルUI（任意）
├─ input/ # 入力データ
└─ output/ # 集計結果
```

## 技術スタック
- Python 3.12
- pandas
- openpyxl
- Streamlit（ローカルUI用）

## 工夫した点
- Excel特有の問題（空行・空列・列名揺れ）を考慮した設計
- 入力ファイルを増やしてもコード修正不要な構成
- ロジック（run.py）と実行方法（bat / UI）を分離し、保守性を向上
- 非エンジニアでも操作できる導線を意識  
- 実行環境や配布時の安定性を考慮し、bat実行を基本とした設計

## 学んだこと
- 業務データは必ずしも整っていないこと
- pandas を用いたデータ統合・集計処理
- Python スクリプトを「業務で使える形」に落とす難しさ
- 配布・運用を意識した設計の重要性
